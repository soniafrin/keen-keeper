🌿 KeenKeeper

📌 Description

KeenKeeper is a simple and interactive relationship tracking app that helps users stay connected with their friends. Users can log quick interactions like calls, texts, and video chats, and visualize their activity through a timeline and stats dashboard.


🚀 Technologies Used
* React.js
* React Router
* Context API (State Management)
* Tailwind CSS
* Recharts (for data visualization)
* React Toastify (for notifications)
  

✨ Key Features

 📞 Quick Check-In System
Easily log interactions (Call, Text, Video) with friends using a clean and intuitive interface.
 📊 Activity Timeline & Filtering
View all interactions in a timeline and filter them by activity type for better tracking.
 📈 Visual Stats Dashboard
Analyze your communication habits using charts and visual insights.
